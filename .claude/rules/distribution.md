# Distribution & Resources

## Installation

**GitHub SSH**: `claude plugin install git@github.com:kingkong0905/my-claude-code.git`
**Local**: Clone to `~/.claude-plugins/`, then install.

## Requirements

**Required**: Claude Code CLI, GitHub CLI (`gh`).
**Optional**: Atlassian OAuth, Python + uv (Serena).
